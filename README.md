## 📖 How to Use

### Browsing Items
- Visit the site and browse all approved items in the **Archives**
- Filter by category using the sidebar (Weapons, Armor, Rings, etc.)
- Use the **search bar** to find specific items
- Click any item card to view its full details

### Creating an Account
1. Click **Login** in the navigation bar
2. Click **Register** and enter a username and password
3. You're in! You can now submit items to the archives

### Submitting an Item
1. Log in to your account
2. Click **Submit** in the navigation bar
3. Fill in the item name, category, and description
4. Optionally paste an image URL (jpg, png, jpeg, webp only)
   - Recommended: upload your image to [imgur.com](https://imgur.com) and paste the direct link
5. Click **Submit to Archives** — your entry will be reviewed by an Operator

### Account Roles
| Role | Permissions |
|----------|------------|
| User     | Browse and submit items |
| Operator | Approve and delete submissions |
| Admin    | Everything + redact items |

##  Tech Stack
- **Frontend:** HTML, CSS, Vanilla JavaScript
- **Backend:** Node.js + Express
- **Database:** SQLite
- **Auth:** JWT (JSON Web Tokens)
- **Hosting:** GitHub Pages (frontend) + Render (backend)
